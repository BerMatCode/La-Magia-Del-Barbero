## 📅 FUNCIONAMIENTO DEL SISTEMA DE RESERVAS

💡 El usuario:
- Selecciona servicio
- Elige profesional
- Escoge fecha y hora
- Ingresa sus datos

📲 Luego:
- Se genera automáticamente un mensaje
- Se abre WhatsApp listo para enviar
- El barbero recibe la reserva al instante
